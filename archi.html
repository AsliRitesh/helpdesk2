Computer Orgranisation and Architecture

Historically there had been two types of computers:-
  1. Fixed Program Computers:
    The functions are very specific and they couldn't be reprogrammed. Ex - Calculator
  2. Stored Program Computers:
    These can be programmed to carries out many different task, applications are stored on them.
    Modern computers arre based on source program concept introduced by John Von Nuemann.
Central Proccessing Unit:
  1. Control Unit: A control unit (C.U) handles all proccess control signals. It direct all input and output flows fetches code for instructions and control how data moves around the system.
  2. Arithematics Logic Unit: The Arithematics Logic unit is that part of the C.P.U that handles all the calculations, the C.P.U mainly.
    Ex - Addition, Substraction, etc.
    It performs logical operations such bitshifting operation, comparisions, etc.
